# Logic Board ⚡ IF → THEN

A minimal, zen‑inspired rule‑based decision engine. Toggle rules ON/OFF, reorder them, set dates, and add a **stop** flag to cancel later rules. Run to see a clean timeline of what fires, sorted by date. Supports both **Gregorian** and **GPC (Global Perennial Calendar)** date display.

## Features

- **IF‑THEN rules** with toggle, date, and optional stop.
- **Stop by list order** – rules fire in the order you see them; a stop rule breaks the chain.
- **Timeline** sorted by date (oldest/newest), with optional GPC conversion.
- **GPC mode** – shows dates in GPC format with day‑of‑week emoji dots.
- **Local storage** – all data persists in your browser.

## GPC (Global Perennial Calendar)

The app includes a full implementation of the **364‑day Global Perennial Calendar** (GPC).  
Learn more about the calendar at **[gpc364.com](https://gpc364.com)**.

The GPC logic is embedded directly – no external dependencies.

## Usage

1. Open `index.html` in any modern browser.
2. Add rules, toggle them on, set dates, and optionally mark "Stop".
3. Press **▶ Run** to see the timeline.
4. Switch between Gregorian and GPC views using the **GPC** button.
